# 2D Platformer Game (C++ / SFML)

A 2D platformer game developed using C++ and the SFML library.  
The game focuses on precise platforming mechanics, traps, and challenging level design.

## 🎮 Features
- Smooth player movement and jumping
- Platform-based level system
- Deadly traps such as spikes
- Moving platforms
- Collision detection system
- Camera scaling for different screen resolutions

## 🛠 Built With
- C++
- SFML (Simple and Fast Multimedia Library)

## ▶️ How to Play
1. Download the **Game.zip** file from this repository.
2. Extract the ZIP file.
3. Run **Game.exe**.

## 🎮 Controls
| Key | Action |
|----|------|
| A / Left Arrow | Move Left |
| D / Right Arrow | Move Right |
| Up arrow | Jump |
| ctrl | Respawn |
